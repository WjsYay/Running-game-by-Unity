# 虚拟现实作业Unity实现跑酷游戏

## 文件使用方法
1. 下载解压文件
2. 打开Unity软件
3. 点击“新项目”后创建一个空白项目
4. 右键该项目，选择在资源管理器中打开
5. 将解压后的文件中的所有内容复制到该项目文件夹中
6. 运行项目
7. 打开Asset文件夹，找到并点击名为“Run”的Scene场景文件，完成场景加载
8. 点击“播放”按钮，即可开始游戏

## 游戏操作
1. 游戏开始后，可以进行模式选择，包括普通模式和无尽模式
2. 玩家使用“WASD”操控角色进行跳跃、翻滚、左转、右转的操作来避开障碍物
3. 当玩家碰到障碍物时，游戏会结束，玩家可以选择重新开始或回到模式选择界面
4. 普通模式规则
    - 普通模式下，游戏完整展示荆轲刺秦王的故事，在一段剧情后开始跑酷游戏
    - 玩家需要坚持指定时间，在指定时间内不要触碰到障碍物
    - 事件结束后，玩家胜利，继续播放荆轲刺秦王之后的故事
5. 无尽模式规则
    - 无尽模式下，玩家没有胜利条件，要持续进行游戏直到碰到障碍物后游戏结束
    - 玩家的目标是尽可能长时间地跑酷，获得更高的分数
